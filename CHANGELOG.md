# Revision summary

The revised technical draft and analysis package address six priority areas:

1. **Code alignment:** the analysis script now implements the models, feature configurations, positive class, repeated cross-validation, nested tuning, class-sensitive metrics, permutation importance and descriptive outputs reported in the dissertation.
2. **Validation and tuning:** the report distinguishes training, inner validation and outer testing; documents fixed primary settings and nested sensitivity grids; and places preprocessing inside pipelines.
3. **Expanded findings:** the chapter now includes sample characteristics, missing-data checks, class distribution, repeated-CV results, confusion matrices, variable importance, Logistic Regression coefficients and nested sensitivity results.
4. **Benchmarks and replication:** naïve baselines and a side-by-side comparison with Cortez and Silva (2008) are included.
5. **Claims corrected:** unsupported ROC-AUC wording was removed; administrative and questionnaire variables are distinguished; interpretability claims are supported with coefficient output and framed cautiously.
6. **Presentation:** the document includes a contents page, lists of tables and figures, consistent headings, a revised reference list, an expanded conclusion and appendices documenting leakage prevention and hyperparameter grids.
